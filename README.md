## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app
   Add your Google Maps Api Key in app.json

   ```json
   "android": {
      "config": {
        "googleMaps": {
          "apiKey": "Your Google Maps Api Key"
        }
      }
   ```

   ```bash
    npx expo start
   ```
