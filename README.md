## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Add your Google Maps Api Key in app.json

   ```json
   "android": {
   "config": {
     "googleMaps": {
       "apiKey": "Your Google Maps Api Key"
     }
   }
   ```

3. To Start the app

   ```bash
    npx expo start
   ```
